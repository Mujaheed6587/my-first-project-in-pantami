My Learning Progress: Git & GitHub Journey

What is GitHub?

GitHub is a online platform that help programmers for hosting and sharing code, enabling collaboration between developers. It is built on top of Git, allowing version control and teamwork on software projects.

What is Git?

Git is a distributed version control system that helps track changes in source code during software development. It allows multiple people to work together, keeping a history of every change.

GitHub Desktop

GitHub Desktop is a graphical application that makes it easier to manage GitHub repositories from your computer. You can commit, push, pull, and manage branches with a simple interface.

How I Created My GitHub Account

  • Visited github.com
  • Clicked "Sign Up" and filled in my details
  • Verified my phone number and address
  • Set up my profile

Repositories

A repository is like a folder for your project. It stores code, files, and tracks changes. I learned to:

  • Create a new repository
  • Clone a repository to my computer
  • Commit changes and push them to GitHub

Hosting on GitHub Pages

  • Enabled GitHub Pages in repository settings
  • Selected my main branch as the source
  • Accessed my hosted site via the provided GitHub Pages link

This allows me to share my project live on the web!
